streams-persist-s3
==============

Write documents to an S3 bucket.

Example writer configuration:

    "s3": {
        "key": "",
        "secretKey": "",
        "bucket": "",
        "protocol": "HTTP",
        "writerPath": "path/",
        "writerFilePrefix": "prefix",
        "chunk": false,
        "region": "us-east-1"
    }
